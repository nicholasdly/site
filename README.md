# nicholasly.com

> My personal website and blog.

## Development

**nicholasly.com** is built with the following tools:

- **Web Framework**: [Astro](https://astro.build/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Deployment**: [Vercel](https://vercel.com/)

### Installation

1. Fork and clone this repository using `git clone`.

2. Install npm packages.

```zsh
npm install
```

3. Run the following command to start a local development server.

```zsh
npm run dev
```

4. Run the following command to format the codebase using [Prettier](https://prettier.io/). Try to do this before every commit.

```zsh
npm run format
```
